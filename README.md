# Smadona Foundation Website

A modern and responsive NGO website developed for **Smadona Foundation**. The platform provides information about the organization's mission, projects, volunteering opportunities, and allows users to submit contact and involvement forms.

## 🌐 Live Website

https://smadhuna-foundation.vercel.app

---

## 📌 Features

- Responsive design for mobile, tablet, and desktop devices
- Modern UI built with Tailwind CSS
- Contact form submission
- Volunteer/Participation form
- MongoDB Atlas database integration
- API routes for storing form data
- Fast deployment using Vercel
- SEO-friendly Next.js application

---

## 🛠️ Tech Stack

### Frontend
- Next.js
- React.js
- Tailwind CSS

### Backend
- Next.js API Routes

### Database
- MongoDB Atlas
- Mongoose

### Deployment
- Vercel
